# Demo Avatar 02 Assets

This directory contains a second bundled demo avatar used to test multi-character switching and non-default avatar canvas sizes.

## Files

- `back-hair.png`
- `front-hair.png`
- `eyes-open-mouth-closed.png`
- `eyes-open-mouth-half.png`
- `eyes-open-mouth-open.png`
- `eyes-closed-mouth-closed.png`
- `eyes-closed-mouth-half.png`
- `eyes-closed-mouth-open.png`
- `eye-highlight.png`

## Canvas size

This demo avatar is normalized to:

```text
900 x 900
```

All expression PNG files in this avatar share the same canvas size and alignment.
