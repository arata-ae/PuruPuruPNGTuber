# Demo Avatar 03 Assets

This directory contains a third bundled demo avatar used to test multi-character switching with layered PNG items.

## Files

- `back-hair.png`
- `front-hair.png`
- `eyes-open-mouth-closed.png`
- `eyes-open-mouth-half.png`
- `eyes-open-mouth-open.png`
- `eyes-closed-mouth-closed.png`
- `eyes-closed-mouth-half.png`
- `eyes-closed-mouth-open.png`
- `items/body.png`
- `items/ribbon.png`
- `items/hairpin.png`
- `default-settings.json`

## Canvas size

This demo avatar is normalized to:

```text
1024 x 1024
```

All expression and item PNG files in this avatar share the same canvas size and alignment.
