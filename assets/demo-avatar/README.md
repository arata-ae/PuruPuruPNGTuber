# Demo Avatar Assets

This directory contains the bundled demo avatar used by PuruPuru PNGTuber.

## Files

- `back-hair.png`
- `front-hair.png`
- `eyes-open-mouth-closed.png`
- `eyes-open-mouth-half.png`
- `eyes-open-mouth-open.png`
- `eyes-closed-mouth-closed.png`
- `eyes-closed-mouth-half.png`
- `eyes-closed-mouth-open.png`
- `items/body.png`
- `default-settings.json`

## Canvas size

This demo avatar is normalized to:

```text
1024 x 1536
```

Other avatars may use different canvas sizes. All expression PNG files within the same avatar must share the same canvas size and alignment.
